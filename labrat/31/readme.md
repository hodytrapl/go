# Лабораторная работа
### Тема: Создание отказоустойчивого обработчика сообщений.
#### Задание 1. Базовая диспетчеризация (15 мин) 
Создайте файл .go.
   1. Напишите две функции-генератора сообщений: newsFeed и socialMedia.
   2. Каждая должна запускаться в горутине и писать в свой канал (строки) с разной периодичностью (используйте time.Sleep).
   3. В main запустите бесконечный цикл for.
   4. Внутри цикла используйте select для вывода сообщений из обоих каналов в консоль с префиксом (например: "Новость: ..." и "Соцсети: ...").
#### Задание 2. Добавление таймаута (Global Timeout) (15 мин) Модифицируйте программу.
   1. Представьте, что мы готовы ждать события суммарно не более 5 секунд на всю работу программы.
   2. Создайте таймер: timeout := time.After(5 * time.Second).
   3. Добавьте case <-timeout: в ваш select внутри цикла.
   4. При срабатывании таймаута программа должна писать "Время вышло, выключаемся" и завершаться (return из main). Внимание: Подумайте, почему time.After нужно создавать ДО цикла, а не внутри case? (Подсказка: если создать внутри, таймер будет сбрасываться на каждой итерации).
#### Задание 3. Неблокирующая проверка (15 мин)
   1. Добавьте в select секцию default.
   2. Пусть она выводит точку . и делает time.Sleep(500 * time.Millisecond).
   3. Запустите. Вы должны увидеть, как программа печатает точки, пока ждет сообщения из каналов. Это имитация "полезной работы" во время простоя.
#### Задание со звездочкой (*): Корректная остановка Реализуйте паттерн Graceful Shutdown.
   1. Вместо time.After для выхода, создайте канал stopChan := make(chan bool).
   2. Запустите отдельную горутину, которая спит 5 секунд, а потом делает close(stopChan).
   3. В главном select обрабатывайте закрытие этого канала. Важно: используйте метку (label), чтобы выйти из бесконечного цикла for.
### Критерии успеха:
  - Сообщения из разных источников перемешиваются.
  - При отсутствии сообщений выводятся точки (если сделано задание 3).
  - Через 5 секунд программа корректно завершается.

# code
```
package main

import (
	"fmt"
	"time"
)

func newsFeed(channel chan<- string) {
	for i := 1; i <= 10; i++ {
		channel <- fmt.Sprintf("Новости из ленты новостей #%d", i)
		time.Sleep(800 * time.Millisecond)
	}
	close(channel)
}

func socialMedia(channel chan<- string) {
	for i := 1; i <= 10; i++ {
		channel <- fmt.Sprintf("блог #%d", i)
		time.Sleep(800 * time.Millisecond)
	}
	close(channel)
}

func main() {
	channel1 := make(chan string)
	channel2 := make(chan string)

	go newsFeed(channel1)
	time.Sleep(1 * time.Second)
	go socialMedia(channel2)

	timeout := time.After(5 * time.Second)

	for {
		select {
		case msg1 := <-channel1:
			println("Новость:", msg1)
		case msg2 := <-channel2:
			println("Соцсети:", msg2)
		case <-timeout:
			println("Время вышло, выключаемся")
			return
		default:
			time.Sleep(500 * time.Millisecond)
			println(".")
		}
	}
}
```

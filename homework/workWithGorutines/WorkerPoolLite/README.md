# Домашнее задание
Тема: Обработка задач с буферизацией (Worker Pool Lite).
Зачем это нужно:
В реальных приложениях часто бывает неравномерная нагрузка. Например, днем приходит 100 заказов в секунду, а ночью 0. Буферизированный канал помогает "сгладить" пики, накапливая задачи, пока исполнитель (worker) занят.
Задача:
Реализовать простую очередь задач.
Файл: homework.go
Требования:
1.	Создайте структуру Task (например, с полями ID int и TimeCreated string).
2.	Напишите функцию-воркер worker(id int, tasks <-chan Task).
    -	Она должна читать задачи из канала в цикле range.
    -	При получении задачи выводить: Воркер [ID] начал задачу [TaskID].
    -	Имитация работы: добавить time.Sleep(time.Millisecond * 500).
    -	После сна выводить: Воркер [ID] закончил задачу [TaskID].
3.	В main:
    -	Создайте буферизированный канал задач taskCh размером 5 или 10.
    -	Запустите одного воркера в горутине.
    -	В цикле (в main) быстро отправьте 10-15 задач в канал. Выводите в консоль сообщение "Отправлена задача X".
    -	После отправки всех задач закройте канал close(taskCh).
    -	Добавьте time.Sleep, чтобы воркер успел всё разгрести.

# Критерии проверки:
1.	Программа компилируется (go run homework.go).
2.	Вы видите, как main очень быстро "выстреливает" первые 5-10 задач (пока буфер не заполнится), а потом начинает притормаживать, ожидая, пока воркер освободит место. Это и есть демонстрация работы буфера.
3.	Все задачи должны быть обработаны, программа корректно завершается.

# code

```
package main

import (
	"fmt"
	"time"
)

type Task struct {
	ID          int
	TimeCreated string
}

func worker(id int, tasks <-chan Task) {
	for task := range tasks {
		fmt.Println("Воркер ", id, " начал задачу ", task)
		time.Sleep(time.Millisecond * 500)
		fmt.Println("Воркер ", id, " закончил задачу ", task)
	}

}

func main() {
	taskCh := make(chan Task, 10)
	go worker(1, taskCh)
	for i := 1; i <= 10; i++ {
		task := Task{
			ID:          i,
			TimeCreated: time.Now().Format("15:04:05.000"),
		}
		fmt.Println("Отправлена задача ", i)
		taskCh <- task
	}
	close(taskCh)
	time.Sleep(time.Second * 3)
}
```
